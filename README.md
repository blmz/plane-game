# ✈️ Godot 小飞机游戏

[](https://www.bilibili.com/video/BV1...)  
[](https://kenney.nl)

使用Godot引擎开发的2D无限跑酷小游戏，玩家操控飞机躲避障碍并收集星星。

## ✨ 核心功能
- 程序化生成随机地形
- 动态分布的尖刺障碍物
- 随机位置的星星收集物
- 随时间递增的速度系统
- 4套飞机/地形主题皮肤
